* 索引
** 常见问题
1. 为什么要使用索引？
   : 快速查询数据
2. 什么样的信息能创建索引？
   : 主键、唯一键、普通键等
3. 索引的数据结构是怎么样的？
   : 生成索引，建立二叉查找树进行二分查找
   : 生成索引，建立 B-Tree 结构进行查找
   : 生成索引，建立 B+-Tree 结构进行查找
   : 生成索引，建立 Hash 结构进行查找
4. 密集索引和稀疏索引的区别？
5. 如何定位并优化慢查询 SQL？
   1) 根据慢日志定位慢查询 SQL
   2) 使用 explain 等工具分析 SQL
   3) 修改 SQL 或者尽量让 SQL 走索引
